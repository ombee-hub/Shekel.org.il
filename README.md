# שק״ל — עיצוב מחדש לאתר

עיצוב מחדש (Redesign) לאתר של **שק״ל – שילוב קהילתי לאנשים עם מוגבלויות**
([shekel.org.il](https://www.shekel.org.il)). אתר **רב-עמודי**, מודרני, חם
ונגיש — שבו הנגישות עצמה היא לב המוצר, בדיוק כמו הייעוד של הארגון.

---

## איך מריצים / צופים

HTML, CSS ו-JavaScript טהורים, ללא תלות בספריות או בכלי בנייה. אפשר לפתוח את
`index.html` ישירות בדפדפן, או להריץ שרת סטטי מקומי:

```bash
npx serve .
# או
python -m http.server 4587
```

```
├── en/                 # גרסת אנגלית מלאה (LTR) — index, about, programs,
│   └── ...             #   news, branches, businesses, donate, contact,

├── js/
│   ├── layout.js       # כותרת + תפריט + פוטר + סרגל נגישות — משותף לכל הדפים
│   └── main.js         # אינטראקציות (תפריט נפתח, נגישות, אנימציות, טפסים)
├── images/             # הלוגו והתמונות (מהאתר הרשמי)
│   ├── logo.svg                       # הלוגו הרשמי (וקטורי)
│   ├── housing / employment / emotional / children / banner-idf
│   │   / accessibility / international .jpg   # תמונות לתחומי הפעילות
│   ├── leisure.jpg                    # תמונת ה-Hero
│   ├── biz-gifts / biz-metsion / biz-studio .jpg   # לוגואים של העסקים החברתיים
│   └── iso9001.jpg, midot-seal.png    # תוויות אמון
└── README.md
```

> **תפריט "תחומי הפעילות"** הוא תפריט נפתח (dropdown) עם חץ שמסתובב — במחשב
> נפתח כתפריט צף, ובמובייל כאקורדיון בתוך המגירה. הקישורים מובילים ל-7 דפי
> המשנה שלמעלה. "דף הבית" הוסר מהתפריט (הלוגו מוביל לדף הבית).
>
> **גרסת אנגלית:** התיקייה `en/` מכילה אתר אנגלי מלא (LTR) המקביל לעברי,
> עם כותרת/פוטר/סרגל-נגישות משלו ([js/layout-en.js](js/layout-en.js)). המעבר
> בין השפות מתבצע דרך הקישור EN / עברית בסרגל העליון. הנכסים (CSS, תמונות,
> main.js) משותפים ונטענים מ-`../`. הקובץ `main.js` מזהה את שפת הדף אוטומטית.

> **כותרת ופוטר משותפים:** כדי לערוך את התפריט, הלוגו או הפוטר — עורכים פעם
> אחת בלבד את `js/layout.js`, והשינוי חל על כל הדפים. כל דף מסמן את עצמו
> בתפריט באמצעות `<body data-page="...">`.

## שפת העיצוב

**קונספט:** חם, אנושי, אופטימי ומעורר אמון — עם נגישות מהשורה הראשונה.

### צבעים (מבוססי המותג הרשמי)
| תפקיד | צבע |
|------|-----|
| מותג ראשי (פטרול-טורקיז) | `#0d7377` / `#15919b` |
| קורל (מהלוגו, כפתורי תרומה) | `#f2674a` |
| מערכת צבעים לתחומי הפעילות | טורקיז · קורל · סגול · ורוד · ענבר · ירוק · כחול |
| רקע חם (קרם) | `#fbf8f3` |

הקורל `#f2674a` והטורקיז נלקחו ישירות מהלוגו הרשמי של שק״ל.

### טיפוגרפיה
- **כותרות:** Rubik · **גוף הטקסט:** Heebo (תמיכה מלאה בעברית, RTL).

### כפתורים
כל הכפתורים **מרובעים עם פינות מעוגלות** (`border-radius` של 14px) — כולל
כפתור הנגישות הצף.

## נגישות (Accessibility)

נבנה לפי תקן **WCAG 2.1 AA** ותקנות הנגישות הישראליות:

- **סרגל נגישות צף** בכל דף: הגדלת/הקטנת טקסט, ניגודיות גבוהה, הדגשת קישורים,
  גופן קריא ועצירת אנימציות — **ההעדפות נשמרות** בין ביקורים (`localStorage`).
- **דף הצהרת נגישות ייעודי** (`accessibility.html`) עם תוכן עניינים, רמת
  ההתאמה, כלי הנגישות ופרטי רכז הנגישות — מקושר מהפוטר ומסרגל הנגישות.
- HTML סמנטי, `dir="rtl"`, ניווט מקלדת מלא, מצבי פוקוס, קישור ״דלגו לתוכן״,
  כיבוד `prefers-reduced-motion` ויעדי מגע גדולים.

## הערות ליישום עתידי
- **התמונות והלוגו** הובאו מהאתר הקיים (`shekel.org.il`) לתיקיית `images/`.
- **תוכן החדשות** הוא תוכן הדגמה — יש להחליפו בעדכונים אמיתיים.
- כפתורי התרומה והסכומים מקושרים כרגע לעוגן — יש לחבר לפלטפורמת התרומות
  (למשל JGive) ולחנות המקוונת.
- הטפסים (יצירת קשר, ניוזלטר) הם הדגמה בצד הלקוח — יש לחבר לשרת/שירות.
- כתובת, טלפון ודוא״ל לקוחים מהאתר הקיים — מומלץ לאמת לפני פרסום.
